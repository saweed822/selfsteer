# Visit Kashmir Tours & Travels

## Overview
A modern, premium tourism website for "Visit Kashmir Tours & Travels". The site features a split hero section for Tour Packages and Car Rentals, detailed package and car listings, and a contact-only booking flow.

## Features
- **Split Hero Home Page:** Immediate choice between Tours and Cars.
- **Tour Packages:** Silver, Gold, and Platinum packages with details.
- **Car Rentals:** Fleet display with pricing and details.
- **Contact Popup:** No booking forms; direct contact via Phone/WhatsApp/Email.
- **Admin Ready:** Data is stored in a PostgreSQL database for future admin dashboard expansion.

## Tech Stack
- **Frontend:** React, Tailwind CSS, Framer Motion, Lucide React.
- **Backend:** Express, Drizzle ORM, PostgreSQL.
- **Design:** Glassmorphism, Premium Dark/Teal/Gold theme.

## Architecture
- `client/`: React frontend (generated).
- `server/`: Express backend.
- `shared/`: Shared types and schema.
- `shared/schema.ts`: Database schema for packages and cars.
